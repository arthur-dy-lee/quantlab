# A股急跌后短期涨跌概率研究

> 标的：**沪深300（CN:000300）** 为主，**上证指数（沪指 sh000001）** 对照
> 区间：近 15 年 **2011-06-16 ~ 2026-06-15**（沪深300 共 3642 个交易日）
> 工具：quantlab `stats` 模块（`drop_gt` / `probability` / `forward_return` / `wilson_interval`）
> 生成日期：2026-06-16

---

## 一、核心结论（先看这里）

1. **单纯"第1天跌>1.5%"，对第2天几乎没有预测力。**
   - 沪深300：第2天上涨 **53.2%**（基准 50.9%，超额仅 +2.3%，置信区间把基准包住）≈ 抛硬币。
   - 上证指数：第2天上涨 58.7%，略强但盈亏比 <1，优势有限。

2. **再加"第2天还跌"（连阴）后，沪深300 第3天反弹概率小幅上升到 58.3%，但仍不显著；上证指数反而没有。**
   - 沪深300：第3天上涨 **58.3%**（N=144，超额 +7.4%，CI 下沿 50.2% 贴着基准）。
   - 上证指数：第3天上涨 **50.5%**（N=105，超额 -1.9%，等于没用）。
   - 两个指数结论相反 → 信号不稳，越细分样本越少越不可靠。

3. **加"低位"过滤后边际仍很弱——之前的 72.4% 主要是未来函数造成的假象。**
   - 改用**纯历史窗口**（信号当天只看过去 3 / 6 个月的高低点，无未来数据、可实盘）重算：
     - 沪深300：过去3个月低位 第3天上涨 **57.1%**（N=77）；过去6个月低位 **62.5%**（N=56）。
     - 上证指数：过去3个月 **45.5%**（N=55，反而偏跌）；过去6个月 55.0%（N=40）。
   - 四个口径的 95% 置信区间都把基准率(~51–52%)包住 → **统计上不显著**，不构成可交易优势。
   - 对照：含未来函数的旧口径曾得 72.4%（N=29），去掉未来数据后掉到 57–63% / 无效，说明那是"事后才知道的低点"造成的偏差。

4. **总体判断**：A股"急跌后短期反弹"在剔除未来函数后**没有稳健、可交易的边际优势**。沪深300 在"过去6个月低位"时第3天反弹概率最高约 62.5%，但仍未达统计显著；上证指数基本无效。叠加越多条件样本越少，且急跌多发生在熊市/股灾，尾部风险大。可作为"低位超跌或有小概率反弹"的**弱先验**，不能当成确定性策略。

5. **附（结论5）：扫描出两类"条件苛刻 + 较高胜率"的策略候选。**
   - **超卖反弹类**：RSI14<25（深度超卖）5日持有胜率 **87.5%**（N=32，上证 81%，≈胜率天花板）——但**强 regime 依赖**，2002–2011 早段只有 52%，仅2011后有效。
   - **动量延续类**：连涨5天 / 涨>2%且RSI>75 / 连跌3天且RSI<30，5日胜率 **67–74%**——且在前半、后半、早段、上证**全部成立**，更稳健。
   - **A股无做空机制 → 看跌信号不可执行，本研究只取做多。**
   - **高胜率 vs 跨周期稳健不可兼得**：想要 85%+ 只能用深超卖（认 regime 风险）；要各时代都稳则上限回到动量类 ~74%。
   - ⚠️ 均为**全样本数据挖掘候选**，存在多重检验/重叠样本偏差，胜率≠收益，需含成本的样本外回测。详见结论5。

6. **附（结论6）：含成本回测后，"胜率"落到"收益"上的真相——是好扳机，不是好全仓策略。**
   - 成本这关轻松过：换手极低（15年仅 17/12 笔），5bps/边成本下每笔只掉 ~0.05pct，胜率 88%→82%。
   - **单押它绝对收益打平甚至略输买入持有**（CAGR 3.4% vs 3.6%）——因为 **97% 时间空仓**，机会太稀缺。
   - **风险调整后完胜**：最大回撤 **-8.8% vs 买入持有 -46.7%**，夏普 **0.77 vs 0.27**，满仓口径年化 220%+。
   - 用法：高确定性**择时/抄底扳机**（空仓资金另作他用），而非能独立致富的全仓策略。详见结论6。

## 二、各结论详细信息

### 口径与方法

- **"跌>1.5%"** = 单日收盘价跌幅 > 1.5%（`close.pct_change() < -0.015`）。
- **"第2天还跌"** = 第2天任意幅度下跌（`pct_change() < 0`）。
- **"第3天涨跌"** = 第3天收盘对第2天收盘的涨跌（`forward_return(1)`）。
- **概率** = 满足条件的样本中"次日上涨"的频率；**基准率(base_rate)** = 全样本无条件次日上涨率，用于衡量超额 edge。
- **置信区间** = 上涨概率的 Wilson 95% 区间。
- **数据来源**：原配置走 akshare 东财接口，但本机网络不可达东财；改用 akshare 的**新浪指数接口**取数，并写入 quantlab 本地库 `data/bars/CN/000300_1d.parquet`，分析全程用 quantlab 自身的 stats 引擎离线计算。

### 结论 2：第1天跌>1.5% → 第2天还跌 → 第3天涨跌

| 指标 | 沪深300 | 上证指数 |
|---|--:|--:|
| 样本 N | 144 | 105 |
| 第3天上涨概率 | **58.3%** | **50.5%** |
| 第3天下跌概率 | 41.7% | 49.5% |
| 基准上涨率 | 50.9% | 52.4% |
| 超额 edge | +7.4% | -1.9% |
| 95% 置信区间 | [50.2%, 66.1%] | [41.1%, 59.9%] |
| 第3天平均 / 中位 | +0.25% / +0.22% | +0.11% / +0.02% |
| 盈亏比 | 0.99 | 1.13 |
| 最近10次（涨/跌） | 4 / 6 | 2 / 8 |

**解读**：加"连阴"筛子后沪深300 略升到 58.3%，但 CI 下沿 50.2% 几乎贴着基准，且上证同口径反而 50.5%（无效）→ 信号不稳、样本变少更不可靠。最近10次两者都偏"第3天继续跌"，与"超跌反弹"直觉相反。

### 结论 4：低位过滤（纯历史窗口，无未来函数，可交易）⭐

> **口径**：在结论2（第1天跌>1.5% → 第2天还跌）基础上，要求信号当天（第2天）处在**过去 N 个月**收盘价区间的**底部 20% 以内**：
> `位置 = (第2天收盘 − 过去N月最低) / (过去N月最高 − 过去N月最低) ≤ 20%`。
> 窗口**只用过去数据（含当日）**，不含任何未来信息 → **可实盘**。N 取 3 个月、6 个月两档。
> （起始期不足整段窗口的信号已剔除，保证口径一致。）

再扫更严的条件后，**RSI14<25、持有5日的 87.5% 基本就是胜率天花板**——加"连跌 / 破布林下轨 / 近250日低位"等确认，或把 RSI 压到 <20，胜率都到不了 88%，只是样本更少（深超卖已把效应吃尽）。代价同样是 regime 依赖：

- 想要 **85%+** 胜率，只能用深超卖（RSI<25）+ 数日持有，且**接受"只在2011后有效"**（早段仅 43–58%）。
- 加"连跌2天"把早段从 52% 提到 65%，是这几个里相对最稳的高胜率口径。

**看跌 / 做空**：A股（指数）**无做空机制，看跌信号不可执行**，本研究只保留做多方向。（附带观察：扫描里也没有稳健的高胜率空头条件，超买 / 连涨后下跌概率最多 ~50%。）

**要点解读**
- **两类策略，取舍不同**：
  - ① **超卖反弹类**（RSI<25、跌+低位）胜率最高（85%+），但**强烈依赖 2011 后的市场结构**（2002–2011 早段仅 42–52%，差于抛硬币）→ 博高胜率，但要认 regime 风险。
  - ② **动量延续类**（连涨5天、强势追涨、连跌+超卖）胜率中等（67–74%），但**在所有子区间和上证都成立** → 更可信、更稳。
- **想稳 → 选动量类；博高胜率 → 选超卖类（认 regime 风险）。**
- 共同硬伤：5日持有 + 信号在急跌 / 单边行情里扎堆 → **重叠样本**使有效 N 远小于名义 N，置信区间偏乐观；且**超卖反弹常发生在暴跌途中，尾部风险大**。**实盘前必须做含成本的样本外回测，把"胜率"换算成"收益 / 回撤"。**

### 结论 6：含成本回测——"胜率"到底能不能落到"收益"上 ⭐

> 用 quantlab `backtest` 引擎，把两个代表信号做成**事件策略**：出现信号 → 持有 5 个交易日（重叠信号顺延）。引擎**次日收盘生效**（防未来函数），按换手计成本。区间近15年。
> 成本档：0 / 5 / 10 bps 每边（5bps/边 ≈ A股 ETF 往返约 0.1%，含佣金，ETF 免印花税）。下表取 **5bps/边**。

**成本敏感性**：换手极低，成本影响微乎其微。RSI<25 在 0/5/10 bps 下 CAGR = 3.53% / 3.41% / 3.29%，每笔胜率 88.2% → 82.4% → 82.4%。**含成本这关轻松过。**

**要点解读**
- **胜率是真的、扛得住成本**：每笔约 +3%，成本只吃掉 ~0.05pct。
- **但"单押它"绝对收益打平甚至略输买入持有**（CAGR 3.4% vs 3.6%）——因为 **97% 时间空仓**，机会一年才 1–2 次，钱大部分时间闲置。
- **风险调整后完胜**：用约 **1/5 的回撤**（-8.8% vs -46.7%）拿到几乎相同收益，**夏普近 3 倍**（0.77 vs 0.27）；满仓口径年化 220%+，资金一旦投进去效率极高。
- **定位**：它是**高确定性的择时 / 抄底扳机**（适合"空仓资金另作他用 / 轮动"或"恐慌时纪律性重仓"），**不是能独立致富的全仓策略**。
- ⚠️ **样本内乐观偏差**：87% 胜率主要存在于 2011 后，本回测区间正好踩在有效期；真正样本外要看未来新数据或 2002–2011（那段仅 52%）。本结论是**含成本的样本内回测**，非严格样本外。
