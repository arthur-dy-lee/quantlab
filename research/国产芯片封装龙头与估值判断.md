## 二、封测龙头:长电科技 vs 蓝思科技(关键区分)

**结论:封装(封测)龙头是长电科技,毫无争议。蓝思科技不做芯片封装。**

"封装"一词有两层含义,两家公司在**完全不同的产业链层级**,严格说不构成竞争:

| | A. 半导体封装/封测(OSAT) | B. 封装基板/材料 |
|---|---|---|
| 干什么 | 晶圆切割、裸芯片装壳、互连、测试 | 提供承载芯片的基板载体 |
| 谁的地盘 | **长电科技** | **蓝思科技**(玻璃基板)切入这层 |
| 产业链位置 | 中游封测服务 | 上游材料 / 基板 |

- **长电**:真正"把芯片封起来"的公司,全球第三大 OSAT。
- **蓝思**:消费电子结构件/玻璃盖板龙头(果链),现切入 **TGV 玻璃基板**——是"给封装环节用的料",不是自己去封芯片。

> 问"封装龙头" → 长电;蓝思只是"先进封装**基板材料**"的新玩家。两者是潜在上下游关系,不是直接对打。

## 三、"超级玻璃" / TGV 玻璃基板赛道

来源:CCTV2 财经报道,"'超级玻璃'或成万亿元新赛道"。

- **TGV(Through Glass Via,玻璃通孔)**:在玻璃基板做垂直互联通孔,作为**有机/硅基板的替代**,成为 AI 芯片封装的理想材料。
- **核心优势**:低损耗、抗翘曲、支持大尺寸生产、成本低于硅基板;解决 AI 芯片尺寸增大导致硅基板易变形的问题。
- **空间预测**:中国工程院院士彭寿预测其或在 **"十五五"末期(约 2030 前后)成为万亿元新赛道**。
- **进度(重要)**:仍处**中试验证**阶段——"预计今年完成首批验证"。**蓝思 TGV 2026 全年无营收贡献,2027 才有望小批量供货**,目前是题材/预期驱动。
- **主要玩家**:
  - 蓝思科技(300433)— 湖南长沙,百万级通孔 0ppm 不通率
  - 凯盛科技(600552)/ 洛阳玻璃 — 中建材系(彭寿院士体系),陕西咸阳激光诱导深刻蚀
  - 沃格光电(603773)等

## 四、芯片封装的"卡脖子"环节

封装中低端国产化率不低,**真正没突破、且高度依赖国外的是最尖端环节,而它们恰恰是高毛利的部分**:

| 卡脖子环节 | 谁在掌控 | 中国现状 |
|---|---|---|
| **CoWoS 级 2.5D 封装**(AI 芯片标配) | **台积电近乎独占**(高良率全栈) | 长电/通富有类 CoWoS,良率/产能差距大 |
| **混合键合 Hybrid Bonding**(亚微米,HBM4/3D) | 台积电(SoIC)、BESI | 国内仅 1–2 家小批量,未规模量产 |
| **TCB 热压键合设备** | **ASMPT(ASM 太平洋)** | 已对部分中国客户停供;美 BIS 列为出口管制重点 |
| **高端 IC 载板(FC-BGA / ABF 载板)** | 欣兴、揖斐电、新光(日台),**ABF 膜 = 日本味之素独家** | 国产几乎空白,最硬卡点之一 |
| **HBM 存储颗粒** | SK海力士、三星、美光 | 长鑫(CXMT)刚起步 |
| **高端封装材料** | 高端塑封料/底填胶/临时键合胶 = 日本 | 国产替代中 |

> **关键洞察**:封装价值链里,毛利最高的不是"封"这个动作,而是 CoWoS / 混合键合这种"把芯片堆起来的尖端工艺"——它被**台积电(代工厂)**拿走了,不是被 OSAT 拿走。长电做的"组装 + 测试"恰是相对标准化、利润薄的那段。

## 五、长电科技:盈利质量与估值

### 2025 年报实数

| 指标 | 数值 | 说明 |
|---|---|---|
| 营收 | 388.7 亿(+8.1%) | 创新高 |
| 归母净利润 | 15.7 亿 | — |
| **净利率** | **≈ 4.0%** | 🔴 非常薄 |
| **毛利率** | **≈ 13–14%** | 🔴 重资产代工典型低毛利 |
| 扣非净利 | **同比 -11%** | 🔴 增收不增利 |
| ROE | ≈ 7% | 🔴 偏低 |
| 资本开支(2025) | 约 90 亿 | 烧钱 |
| 自由现金流 | 约 -44 亿 | 🔴 为负 |
| 先进封装收入 | 270 亿(占比 ~70%) | 良率爬坡反拖累利润 |

### 全球地位
- 全球第三大 OSAT:日月光(~27%)、安靠 Amkor(~14%)、长电(~10%)。**国内第一,但不垄断**。
- 最高端 AI 封装(CoWoS 类)主导权在台积电,长电是追赶者。

### 估值与判断
- 市值约 1088 亿 / 净利 15.7 亿 → **PE ≈ 70 倍**。
- 用 70 倍 PE 买一个净利率 4%、ROE 7%、FCF 为负的重资产公司,**纯看当前盈利明显贵、撑不住**。
- 市值是**"AI 先进封装放量"的成长预期**撑起来的,不是当前基本面。属高 β、靠业绩兑现 + 估值扩张驱动的标的。

> **反直觉总结**:封装里真正"高利润 + 卡脖子"的部分(CoWoS、混合键合、ABF 载板),长电基本没吃到;它吃到的是规模大但利润薄的部分。它的高市值,本质是在为"未来能不能爬上那些高利润环节"付钱。

## 六、寒武纪:估值与"配不配市值"

### 与长电对比:盈利质量两个极端

| | 寒武纪(688256) | 长电(600584) |
|---|---|---|
| 2025 营收 | 64.97 亿 **(+453%)** | 388.7 亿 (+8%) |
| 2025 归母净利 | 20.59 亿 **(+555%)** | 15.7 亿 |
| **毛利率** | **≈ 55%** 🟢 | ≈ 13% 🔴 |
| **净利率** | **≈ 39%** 🟢 | ≈ 4% 🔴 |
| 2026Q1 | 营收 28.85 亿(+160%)、净利 10.13 亿(+185%) | — |
| 市值 | 约 6700–8300 亿 | 约 1088 亿 |
| PE(TTM) | **约 300 倍** 🔴 | ~70 倍 |

> 反差:长电是"低毛利重资产、PE 还 70 倍"(贵在没利润);寒武纪是"高毛利高增长、但 PE 300 倍"(贵在价格太高)。

### 为什么"单看 PE"会误判寒武纪
PE 在这几种情况失效,寒武纪全中:① 净利刚扭亏(分母极小/剧变);② 高速成长期(今年 300 倍,2028E 才 ~32 倍);③(周期股顶部);④ 一次性损益。

### 结论
- 基本面质量**远好于长电**(高毛利、高增长、国产算力替代趋势确定)。
- 但**价格把乐观预期透支得更彻底**:现价已透支未来 2–3 年业绩,容错率极低。
- 风险:增速一旦不及预期(华为昇腾分流、客户集中、政府订单回款周期长、应收账款大)→ 戴维斯双杀。
- **"配不配市值"取决于:你信不信它 2026–2028 能连续翻倍式增长并兑现。** 属"好公司贵",不是"烂公司贵"。

## 七、估值判断方法论(通用框架)

### 第 1 步:按公司阶段"选对尺子"(最关键)

| 公司阶段 | 该用什么 | 例子 |
|---|---|---|
| 亏损 / 早期 | **PS(市销率)**、TAM×渗透率、EV/Sales | 早期寒武纪、生物药 |
| **高增长盈利** | **PEG、前瞻 PE、PS** | 现在的寒武纪、英伟达 |
| 成熟稳定 | PE、股息率、DCF | 茅台、长江电力 |
| 重资产 / 周期 | **PB、EV/EBITDA** | 长电、钢铁、面板 |
| 银行 / 地产 | PB + ROE | 招行 |

> 给长电用 PE、给寒武纪用当年 PE,都是"拿错尺子"。

### 第 2 步:看四层

1. **成长性**:增速 + 持续性 → **PEG = PE ÷ 净利增速**。<1 便宜、≈1 合理、>1 偏贵。
2. **盈利质量**:毛利 / 净利 / ROE / **现金流 / 应收账款**。
3. **空间与格局(护城河)**:TAM、市占率、竞争对手。
4. **横纵对比 + 预期差**:对自身历史 PE/PS band、对同业;**当前价透支了几年业绩?容错空间还剩多少?**

### 一句话方法论
> 判断估值不是"PE 高 = 贵、低 = 便宜",而是:① 先按阶段选对尺子;② 把"价格(PE/PS)"和"成长速度(PEG)"放一起看;③ 用盈利质量、行业空间、竞争格局校验;④ 最后问——现价透支了未来几年?**PE 只是其中一把尺子,而且常常是最容易骗人的那把。**

3. **寒武纪 2028E PE(~32 倍)= 今天苹果/亚马逊的水平**:需先兑现整整 3 年高增长,估值才"降"到今天苹果的常态——现价已把 2028 才该有的业绩提前定价。

## 九、核心结论 TL;DR

- **封装(封测)龙头 = 长电科技**;蓝思科技是"先进封装玻璃基板材料"的新玩家,赛道不同。
- **高利润 + 卡脖子的尖端封装(CoWoS / 混合键合 / ABF 载板)被台积电与日台设备材料商掌控,长电没吃到。**
- **长电**:低毛利(13%)、低净利率(4%)、FCF 为负的重资产生意,PE 70 倍靠"成长预期"撑——当前基本面撑不起市值。
- **寒武纪**:基本面质量好得多(毛利 55%、净利 39%、三位数增长),但价格透支更狠;不能用当年 PE 判断,需前瞻 PE + PS + PEG + 兑现风险综合看;PS 100 倍全场最贵。
- **估值与基本面最匹配的是英伟达、台积电**(PEG ≤ 1);苹果/谷歌"优质但偏贵";**寒武纪 / 特斯拉是"好赛道 + 极高溢价 + 靠故事撑估值",赌的是增长兑现速度。**
- **方法论**:先按阶段选对尺子,再把价格与成长速度(PEG)合看,用盈利质量/空间/格局校验,最后问"透支了几年、容错还剩多少"。PE 是最容易骗人的单一指标。
