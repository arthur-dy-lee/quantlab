# 市场温度计（Market Thermometer）设计

> 目标：用一组指标综合判断「股市整体高不高」，输出 0–100 的温度分。
> 不是单一巴菲特指数，而是**估值 + 利率 + 流动性 + 情绪 + 供给**五维加权。
> 中股、美股使用**不同权重**；并拆成**顶部温度**与**底部温度**两张表（顶/底不对称）。

---

## 1. 核心方法论

每个指标统一走三步：

1. **取原始序列** → 例如沪深300股息率、中证全指PB、两融余额…
2. **转历史百分位（0–100）** → 复用现有 `macro_source.historical_percentile()`，
   即「当前值在截至当时的全部历史中的分位」。窗口：A股 10–15 年，美股 20–30 年。
3. **方向对齐** → 统一成「分值越高 = 越贵/越热」（顶部表）或「越高 = 越便宜/越恐慌」（底部表）。
   ERP、PB、破净率等是「越低越贵」的，需先反向。

加权求和 → 该市场的温度分。

**预处理约定**
- 百分位前先 **winsorize（掐头去尾 1%）**，避免 2015 配资、2008 这类极端值扭曲。
- 数据频率对齐到**日度**：慢变量（CAPE、IPO、M2）月/季度，用最近值**前向填充**。
- 缺某指标时，按**剩余指标权重归一化**，不让缺数据拖低总分。

## 2. 标的与数据源（已决定）

### 2.1 指数标的

| 用途 | A股 | 美股 |
|------|-----|------|
| **主温度 / 全市场口径**（PB、破净率、PE分位、换手率） | **中证全指**（覆盖几乎全部A股） | **标普500** |
| **ERP / 股债性价比** | **沪深300**（股息率稳、核心资产） | 标普500 |

> 默认展示**中证全指温度**；ERP 这一项单独用沪深300。

| 市场 | 数据源 | 说明 |
|------|--------|------|
| A股 | **akshare**（首选，零门槛免注册） | PE/PB分位、破净率、两融、换手、IPO、国债收益率、股息率、M2/社融、北向 |

| 美股 | **FINRA** | margin debt（月度） |

## 3. 指标清单与测量维度

| # | 指标 | 维度 | A股取数 | 美股取数 |
|---|------|------|---------|----------|
| 1 | **CAPE / PE历史分位** | 估值锚 | 中证全指 PE 分位（A股 CAPE 失真，用 PE 分位代） | Shiller CAPE |
| 2 | **股债性价比 ERP / Fed Model** | 估值×利率（核心） | 沪深300盈利收益率 − 10年期国债 | 标普盈利收益率 − 10Y |
| 3 | **PB / 破净率** | 估值底（资产） | 中证全指 PB + 全市场破净占比 | 标普 PB |
| 4 | **M2增速 / 流动性** | 流动性 | M2同比 + 社融 + 北向资金 | M2同比（FRED M2SL） |
| 5 | **情绪 / 拥挤**（复合） | 情绪 | 两融占流通市值×0.6 + 换手率分位×0.4 | margin debt×0.6 + VIX反向×0.4 |
| 6 | **IPO + 再融资 / 总市值** | 供给抽血 | akshare 新股 + 增发 | Renaissance/手动（v2） |

## 4. 权重设计

### 4.1 综合温度（单一，参考用）

| 指标 | 美股权重 | A股权重 |
|------|:---:|:---:|
| CAPE / PE分位 | **28%** | 18% |
| ERP / Fed Model | **30%** | **26%** |
| PB / 破净率 | 6% | **18%** |
| M2 / 流动性 | 12% | 14% |
| 情绪 / 拥挤 | 16% | 16% |
| IPO / 再融资 | 8% | 8% |
| **合计** | **100%** | **100%** |

**中美差异逻辑**
- 美股 **CAPE+ERP 占 58%**（盈利+利率主导，理性、均值回归强）。
- A股 **ERP+PB/破净 占 44%，情绪+流动性+供给 占 38%**（更靠资金面和情绪；CAPE 降权用 PE 分位）。

### 4.2 顶部温度（过热，抓泡沫）—— 重仓杠杆/换手/IPO/CAPE/低ERP

| 指标 | A股 | 美股 |
|------|:---:|:---:|
| CAPE / PE分位（高=贵） | 16% | 26% |
| ERP（低=贵） | 16% | 22% |
| PB / 破净率 | 4% | 4% |
| M2 / 流动性 | 12% | 12% |
| 情绪 / 拥挤（两融+换手 / margin+VIX低） | **32%** | **22%** |
| IPO / 再融资供给 | **20%** | 14% |
| **合计** | **100%** | **100%** |

### 4.3 底部温度（恐慌，抓机会）—— 重仓破净/高ERP/低PB

| 指标 | A股 | 美股 |
|------|:---:|:---:|
| CAPE / PE分位（低=便宜） | 16% | 24% |
| ERP（高=便宜） | **30%** | **32%** |
| PB / 破净率 | **28%** | 12% |
| M2 / 流动性（宽松） | 12% | 14% |
| 情绪 / 拥挤（去杠杆 / VIX高=恐慌） | 10% | 14% |
| IPO / 再融资（停发=底部信号） | 4% | 4% |
| **合计** | **100%** | **100%** |

**顶/底不对称逻辑**
- **顶部专属信号**：两融、换手率、IPO 放量、CAPE 极值 → 顶部表重仓。
- **底部专属信号**：破净率、ERP 走高、PB 走低 → 底部表重仓。
- 牛市顶看顶部表、熊市底看底部表，比单一温度计更灵。

---

## 5. 输出设计

- **两个 0–100 仪表盘**：顶部温度、底部温度。
- **综合指针**：`净温度 = 顶部温度 − 底部温度`（范围 −100 ~ +100），>+40 偏泡沫、<−40 偏机会。
- **分项贡献条**：每个指标当前百分位 + 对总分的贡献，便于溯源「为什么热」。
- **历史回看**：温度分叠加指数走势，验证历史拐点（2007/2015 顶、2008/2018/2024 底）。
